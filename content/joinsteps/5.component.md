`components > Memebrs.vue`を開くと、scriptタグ内に`members`を定義している記述があります。  
前回の人の記述の続きから、以下をコピペしてプロパティに自分の情報を記述しましょう。  

```JavaScript

{
  id: 'ここに GitHub ID',
  displayname: 'ここに表示名',
  bio: 'ここに簡単な自己紹介'
},

```

書き終わったら保存して、次のステップで`commit`します。
